# Comandos de consola

## Crear directorios

    mkdir directorio1 directorio2 directorio3

##  Crear archivos vacios

    touch archivo1 archivo2

## Limpio consola

    clear

## Listar directorios
 
    ls 

## Cambio de directorio (con tabulador me va ir suguiriendo)

    cd <directorio>

## Salir de un directorio

    cd ..
